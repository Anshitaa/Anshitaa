My work sits at the intersection of **Agentic AI**, **Data Engineering**, and **MLOps**. I don't just prototype — I architect, deploy, and monitor systems that handle real data at scale.

### [Docu-Sync](https://github.com/Anshitaa/DocumentSync) — Multimodal GenAI Pipeline *(IBM Watsonx Hackathon)*
> *Gemini 1.5 Flash (Vision) · RAG · text-embedding-004 · OpenCV · Flask · Docker*

A **4-stage multimodal Generative AI pipeline** that watches UI changes and automatically opens a GitHub PR with updated documentation.

- **Stage 1 (Detection)** — SSIM (OpenCV) for pixel-level diff + Gemini Vision for semantic understanding
- **Stage 2 (RAG)** — chunks README by markdown header, embeds with text-embedding-004, retrieves top-3 relevant sections via cosine similarity
- **Stage 3 (Doc Gen)** — Gemini 1.5 Flash generates structured JSON output with confidence score (0 to 1) and reasoning
- **Stage 4 (GitHub PR)** — PyGithub opens a pull request automatically
- **Eval suite** scoring pipeline quality across 11 dimensions
